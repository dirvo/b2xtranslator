SUB: Subscriber (91h)

The SUB record contains information about the publisher/subscriber feature.  This record can be created only by Excel for the Macintosh.  However, if Excel for any other platform encounters the SUB record in a BIFF file, it leaves the record in the file unchanged, when the file is saved.
