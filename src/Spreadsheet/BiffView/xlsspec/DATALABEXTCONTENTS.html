DATALABEXTCONTENTS: Chart Data Label Extension Contents (86Bh)

Introduced in Excel 10 (2002) this BIFF record is an FRT record for Charts.  This record holds Chart Data Label extensions which are new for Excel 10.  Specifically, it indicates which chart information is contained in the data labels, and the separator string used between parts of the data label.
