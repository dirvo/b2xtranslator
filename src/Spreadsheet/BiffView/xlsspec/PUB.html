PUB: Publisher (89h)

The PUB record contains information about the publisher/subscriber feature.  This record can be created only by Microsoft Excel for the Macintosh.  However, if Excel for any other operating environment encounters the PUB record in a BIFF file, it leaves the record in the file, unchanged, when the file is saved.
