GCW: Global Column-Width Flags (ABh)

This record contains an array of 256 flag bits, where each bit represents a column on the sheet. If a bit is true, it means the corresponding column has the Use Standard Width option turned on. If a bit is false, it means that the column has the Use Standard Width option turned off. If the Standard Width measurement was changed (that is, if it is no longer the default), Excel writes a STANDARDWIDTH record.
