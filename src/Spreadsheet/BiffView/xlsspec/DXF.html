DXF: Differential XF (88Dh)

The DXF record is used to describe formatting properties that are used by various features to modify cell formatting. BIFF8 records such as CF and SXDXF still persist DXF information inline (possibly followed by an xfexts structure to round trip post BIFF8 formatting). However new feature records (e.g. TABLESTYLEELEMENT ) refer to DXF s with a dxfID which is the index of this corresponding DXF record as saved on disk.
