Sheet type:
00h = worksheet or dialog sheet
01h = Excel 4.0 macro sheet
02h = chart
06h = Visual Basic module

 

A Visual Basic procedure can set the Visible property of a sheet to create a very hidden sheet.  A very hidden sheet can be made visible again by a Visual Basic procedure, but there is no way to make the sheet visible through the user interface of Excel.
