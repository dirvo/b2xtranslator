FEAT12: Shared Feature 12 Record (878h)

Introduced in Office Excel 2007 the Shared Feature 12 ( FEAT12 ) record has the same data layout as the FEAT11 (Shared Feature 11) record with the benefit that Office Excel 2003 doesnâ€™t understand the contents of a FEAT12 record. For instance this is used to save a Table when Office Excel 2007 features make the Table unrecognizable and/or unusable in Office Excel 2003.
