CRASHRECERR: Crash Recovery Error (865h)

Introducted in Excel 10 (2002) this record is an FRT record. This BIFF record is used by Excel’s crash recovery mechanism to report to the user errors detected during crash recovery of the workbook (BIFF file).  The meaningful info in this record is the error message text contained in the cch and rgch fields, with possible CONTINUE record(s) following, if the error message text spans more than one BIFF record.
