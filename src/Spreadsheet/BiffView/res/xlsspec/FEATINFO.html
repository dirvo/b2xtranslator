FEATINFO: Shared Feature Info Record (86dh)

Introduced in Excel 10 (2002) the FEATINFO record describes shared feature definitions and options or handling instructions for various shared features, such as Protection, Factoid etc.

There should be one FEATINFO record for each shared feature type that exists in a workbook.  That means, this block of data describes the common handling instructions of a shared feature of the same type regardless of whether the features are on the same or different sheets of a workbook.
