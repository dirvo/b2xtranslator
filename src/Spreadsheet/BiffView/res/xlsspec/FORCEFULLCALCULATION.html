FORCEFULLCALCULATION: Force Full Calculation Mode (8A3h)

This record stores the status of the forced calculation mode for the current workbook.

Offset

Name

Size

Contents

rt

2

Record type; this matches the BIFF rt in the first two bytes of the record; =08A3h

(Reserved)

8

Currently not used, and set to 0

16

fNoDeps

4

=1 if the workbook is in the forced calculation mode, in which case dependencies are ignored, and all worksheets are marked to calculate fully every time the calculation is triggered. Stored as 4-byte integer value
