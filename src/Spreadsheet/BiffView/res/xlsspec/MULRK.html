MULRK: Multiple RK Cells (BDh)

The MULRK record stores up to the equivalent of 256 RK records; the MULRK record is a file size optimization.  The number of 6-byte RKREC structures can be determined from the ColLast field and is equal to (colLast-colFirst+1) .  The maximum length of the MULRK record is (256x6+10)=1546 bytes, because Excel has at most 256 columns.  Note : storing 256 RK numbers in the MULRK record takes 1,546 bytes as compared with 3,584 bytes for 256 RK records.
