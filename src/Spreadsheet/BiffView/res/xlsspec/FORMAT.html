FORMAT: Number Format (41Eh)

The FORMAT record describes a number format in the workbook.

All the FORMAT records should appear together in a BIFF file. The order of FORMAT records in an existing BIFF file should not be changed.  It is possible to write custom number formats in a file, but they should be added at the end of the existing FORMAT records.
