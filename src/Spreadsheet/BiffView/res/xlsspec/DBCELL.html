DBCELL: Stream Offsets (D7h)

The DBCELL record stores stream offsets for the BIFF file. There is one DBCELL record for each block of ROW records and associated cell records.  Each block can contain data for up to 32 rows.  For more information about the DBCELL record, see “Finding Cell Records in BIFF Files EXB0250 ”.
