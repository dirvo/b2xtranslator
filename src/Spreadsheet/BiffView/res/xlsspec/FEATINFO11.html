FEATINFO11: Shared Feature Info 11 Record (873h)

Introduced in Excel 11 (2003) the FEATINFO11 record describes shared feature definitions and options or handling instructions for various shared features in Excel 2003 such as Protection, Factoid (SmartTag) and Table.

This record is similar to FEATINFO in Excel 2002, but has a better built in mechanism for round-tripping through earlier versions of Excel. Hence, starting from Excel 2003, FEATINFO is no longer used for saving Shared Features handling instructions.  Instead, this FEATINFO11 should be used in its place.

Likewise, there should be one FEATINFO11 record for each shared feature that exists in a workbook.  This means, this block of data describes the common handling instructions of a shared feature of the same type regardless of whether or not the features are on the same or different sheets of a workbook.
